Classic

The classic Minesweeper experience. To win, reveal all safe tiles without clicking on any mines. Numbered tiles indicate the number of mines hidden in the adjacent eight tiles.

Multimines

Minesweeper, but with up to four mines per tile.

Omega

Minesweeper, but also with negative mines. All tiles adjacent to '0's are automatically revealed if they contain no mines (either positive or negative). This means all unrevealed tiles adjacent to '0's are guaranteed to contain a mine. You may only chord on number tiles with exactly one unrevealed adjacent tile remaining.

Liar

Minesweeper, but number tiles "lie" by displaying numbers as one off from the actual value. You may only chord in the following trivial cases:

  • The number of adjacent flags exceeds the value of the number tile
  • When only one adjecent tile is hidden, and the number of adjacent flags is exactly one greater than the value of the number tile

  • [Space]: Restart game
  • [Left Click]: Reveal tile
  • [Right Click]: Flag tile
  • [Left+Right Click]: Chord on number tile
